CAIO HENRIQUE

Minha jornada começou no mundo do design, onde descobri minha paixão por criar interfaces bonitas e funcionais. Mas sempre tive curiosidade em entender como tudo funcionava nos bastidores - foi assim que mergulhei no universo da programação e me apaixonei por desenvolvimento web.


Atualmente, meu foco principal está no front-end, onde trabalho diariamente com HTML, CSS e JavaScript para criar experiências digitais fluidas e responsivas. Mas também tenho conhecimentos em back-end com PHP, Python e Java, além de experiência com bancos de dados como SQL, MySQL e MongoDB. Para manter tudo organizado, aplico metodologias ágeis em meus projetos.


O que mais me motiva é transformar ideias em produtos reais que as pessoas possam usar e aproveitar. Seja um site institucional, uma landing page ou uma aplicação mais complexa, busco sempre aliar a solidez técnica com uma experiência do usuário excepcional.


Quando não estou codando, você me encontra lendo algum livro de tecnologia ou ficção, jogando videogame (sou um grande fã!) ou tomando aquele café que todo dev precisa para render.


Se você está procurando um desenvolvedor front-end com visão completa do projeto e atenção aos detalhes, adoraria conversar! Dá uma olhada no meu trabalho e vamos criar algo incrível juntos.
